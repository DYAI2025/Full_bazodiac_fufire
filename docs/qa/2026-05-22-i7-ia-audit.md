5 Berechnungs-Lanes nebeneinander (`bazi`, `western`, `wuxing`, `fusion`, `houses`) + 5 weitere. Mobil-Viewport schwer scrollbar.

### 2. Orphan-Routes
- `/transit-calendar` — 0 inbound, kein Nav-Tab, nur via direkter URL
- `/method` — 0 inbound, nur via SecondaryNav-Tab erreichbar

### 3. Hidden Deep-Dive-Cluster
`/love`, `/personality`, `/career-finance` querverlinken sich gegenseitig, kein Eingang aus Main-Nav. Entweder historisch obsolet oder bewusste Sekundär-Oberfläche — Klärung nötig.

### 4. Self-Links (5 Pages)
DailyPage, DashboardPage, LovePage, PersonalityPage, CareerFinancePage linken auf sich selbst. Wahrscheinlich Tippfehler oder Refactor-Reste.
